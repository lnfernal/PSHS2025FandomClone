FandomClone, the Wiki for Fandoms!

FandomClone is a website designed by Gabee De Vera. It is based off of the Fandom Wiki. The website was first worked on when he was 13 years old.

Development

FandomClone was developed on HTML, CSS and JavaScript. No external Libraries or Frameworks were used in programming it.

Visual Studio Code

FandomClone was originally developed on Gabee's local text editor, VSCode.


During this time, many features were developed, such as a search bar and navigational bars. There were also three fandoms and a considerable number of articles. The only thing lacking was that the code resided in the local machine, and not somewhere in the web.

GitHub

The FandomClone is currently stored as a GitHub repo. This allowed for it to be hosted as a github page, and also allowed collaboration. As of July 30 though, Gabee is the only one working on it so far.
